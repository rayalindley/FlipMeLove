
# Flip Me Love

A fun little Electron app for this Valentine's Day. Click anywhere to start, match the cards, and read some sweet messages.


## Installation

For Developers:

```bash
  git clone https://github.com/yourusername/FlipMeLove.git && cd FlipMeLove
  npm install
  npm start 
```
